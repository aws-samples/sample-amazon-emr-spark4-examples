# Amazon EMR Spark 4 Examples

This repository contains sample code and examples for using Apache Spark 4 on Amazon EMR. This package is structured based on the following directories:

- **pyspark** - PySpark examples (e.g., IoT monitoring)
- **scala3** - Scala 3 examples (e.g., Scala 2 → 3 migration patterns, KDS + Protobuf)
- **spark-sql** - Spark SQL examples
- **spark-connect** - Spark Connect examples
